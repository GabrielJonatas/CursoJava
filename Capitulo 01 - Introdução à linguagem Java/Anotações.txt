Java SE(Standard Edition) LTS(Long Term Support) <- Versão mais adotada

Java -> linguagem de programação (regras sintáticas) e 
plataforma de desenvolvimento e execução (Bibliotecas[API] e ambientes de execução)

Aspectos notáveis

- Código compilado para bytecode e executado em máquina virtual (JVM)
- Portável, segura, robusta
- Roda em vários tipos de dispositivo
- Domina o mercado corporativo desde o fim do século 20
- Padrão Android por muitos anos

Edições

Java ME - Java Micro Edition (dispositivos embarcados e móveis IOT)
Java SE - Java Standard Edition (Desktop e servidores)
Java EE - Java Enterprise Edition (Aplicações corporativas)

Plataforma Java SE

- JVM - Java Virtual Machine (Máquina virtual do Java, necessário para executar sistemas Java)

Modelo de execução Java

-Compilação -> bytecode (Representação intermediária) -> 
-Compilação just-in-time (máis rápido que a interpretação) JVM -> Código de máquina

Java (linguagem orientada a objetos)

-Todo código em Java está dentro de classes
-package = agrupamento lógico de classes relacionadas
-Módulo = agrupamento lógico de pacotes relacionados
-Runtime = agrupamento físico
-Aplicação = Agrupamento de módulos relacionados
